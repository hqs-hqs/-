# 项目名称：在线小游戏平台

## 项目描述
基于 Vue 3 + Vite + Spring Boot 的全栈在线游戏平台，包含五子棋、数独、打地鼠、坦克大战等多款经典小游戏，支持用户登录注册、游戏数据持久化、商城皮肤系统等功能。

## 技术栈
- **前端**：Vue 3 + Vite + Vue Router + Axios
- **后端**：Java Spring Boot + MySQL
- **构建工具**：Vite
- **HTTP 客户端**：Axios

## 核心功能模块
- **用户认证系统**：实现注册、登录、密码重置功能，支持 API 与 localStorage 双模式数据存储
- **游戏大厅**：多款经典游戏入口，集成多语言（中英文）切换功能
- **游戏模块**：
  - 五子棋：双人对弈，支持禁手规则
  - 数独：经典数独，支持多难度级别
  - 打地鼠：计分系统，限时挑战
  - 坦克大战、跑酷、碰碰车等
- **商城系统**：皮肤购买、金币管理、用户道具仓库
- **音效系统**：全局鼠标点击音效、游戏内独立音效支持

## 负责模块
- 用户登录注册模块设计与实现
- 游戏大厅界面开发与多语言适配
- API 接口封装与请求拦截器实现
- 游戏数据持久化存储方案设计
- 商城系统前端界面开发

## 技术要点
- 运用 Vue 3 Composition API 进行组件化开发，提高代码复用性
- 使用 Vue Router 实现单页面应用路由管理
- 通过 Axios 拦截器统一处理 Token 认证与错误响应
- 采用 localStorage 实现离线数据存储，支持无后端运行模式
- 实现响应式布局，适配 PC 与移动端设备
